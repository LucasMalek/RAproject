Como você deseja entrar com o Banco de Dados?

Tutorial de Utilização

1. Criando o banco de dados

Após selecionar "Criar banco de dados", uma interface de criação de tabelas aparecerá. Para adicionar ou remover atributos de suas tabelas, basta clicar no símbolo "+" ou "-" que aparecem ao lado do último atributo existente na primeira linha. Caso queira adicionar ou remover tuplas, basta clicar no "+" ou "-" que aparecem no lado esquerdo da ultima tupla existente. Para adicionar mais tabelas, basta clicar no símbolo "+" que aparece ao lado da última aba de Tabela existente.

2. Selecionando e Inserindo Operadores

Para começar, utilize o teclado virtual na parte superior da tela para selecionar os operadores que deseja inserir. Basta clicar no operador desejado, e ele será adicionado automaticamente ao campo de consulta. Para inserir informações como atributos ou relações, clique nos espaços marcados e insira os dados necessários.

3. Concatenando Operadores

Após inserir um operador, você pode concatenar outros operadores para construir consultas mais complexas. Selecione um operador já inserido para conectá-lo com outros, e clique nos novos operadores desejados.

4. Configurando as Tabelas

No canto esquerdo na aba 'Consulata', Você encontrará o campo de relações. O campo de relações comporta "clusters" para cada tabela no banco de dados de entrada. Cada tabela conta com alguma opções de configurações que podem ser vistas no canto superior direito do cluster da tabela. O botão "↗" permite visualizar a tabela inteira. O botão "+" permite adicionar novos atributos. O botão "-" remove os atributos selecionados. O botão "↻" permite atualizar dados da tabela. O botão "X" apaga a tabela.

5. Executando Consultas

Após definir sua consulta, clique no botão "Consultar" no canto inferior direito para executá-la. Você será redirecionado para a aba "Resultados" caso a consulta seja bem sucedida.

6. Histórico de Consultas

O histórico de consultas é exibido na aba "Resultados". Você pode selecionar uma consulta antiga para revisitar seu resultado. Para remover uma consulta do histórico, clique no botão de lixeira ao lado da consulta selecionada.

7. Criando sequências de operações

O operador 'Atribuição' pode ser utilizado para sequências de operações de mais de uma linha. Basta selecioná-lo para criar uma nova linha de operações na consulta.

8. Visualizando Resultados Intermediários

Para visualizar os resultados intermediários de cada atribuição em sua consulta, basta clicar no botão de referência da linha da atribuiçao no canto direito do título "Resultado".

Relações
Dicas:
1. O operador 'Atribuição' pode ser utilizado para sequências de operações de mais de uma linha.
2. Não lembra como utilizar um operador? Deixe o mouse sobre o operador para uma descrição.
3. Dúvidas de como utilizar a ferramenta? Siga para o tutorial.
4. Valores de atributos do tipo string devem ser escritos entre aspas simples!
